## :checkered_flag: Getting starter


```bash

# Clone the project

$ git clone https://github.com/joaogomesdev/contentlayer.git

# Go to project directory

$ cd contentlayer

# Install the dependencies

$ npm install 

# Run the project

$ npm run dev 

# The server will initialize on <http://localhost:3000/>

```
